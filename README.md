# EHR Medallion Pipeline

End-to-end EHR data pipeline on Databricks using Medallion architecture 
(Bronze → Silver → Gold), schema validation, and an AI agent.

## Status
🔨 Under active development — README will be updated as the project progresses.

## Stack
- Databricks (DBR 14.3 LTS+)
- Delta Lake / Delta Live Tables
- Unity Catalog
- PySpark
- Python 3.10+
